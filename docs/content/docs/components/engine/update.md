# engine update

`Update engine binary`

## Synopsis

`Update engine binary`

```
engine update [flags]
```

## Examples

```
engine update --from-github
```

## Options

```
      --api string    Update binary from a CDS Engine API
      --from-github   Update binary from latest github release
```
