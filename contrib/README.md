# CDS Contrib

Here you'll find extensions ready to use with [CDS](https://github.com/ovh/cds).

CDS support several kind of extensions:

- Actions & Plugins
- worker-models
- workflow-templates
- µServices

See [CDS documentation](https://github.com/ovh/cds) for more details.
